# CppStringJoiner

## Overview
CppStringJoiner is a C++ program designed to concatenate a list of strings into a single string, separated by a specified delimiter. This utility is particularly useful for creating comma-separated values (CSV) or any other delimited text format.

## Features
- Concatenate multiple strings using a custom delimiter.
- Easy to use with a simple command-line interface.
- No delimiter added at the end of the last string.

## Getting Started
To use CppStringJoiner, follow these steps:

1. Clone the repository to your local machine.
2. Compile the source code using a C++ compiler
3. Run the executable:
4. The program will output the joined string to the console.
